Package: wooga.gradle.unity

[Groovy] Interface UnityActionConvention

  • Base type for all unity action classes.

    Every Unity Action/Task implements this interface to be able to set and retrieve basic configuration properties

      • java.lang.String getLogCategory()

        Returns the log category.

        The log category property is used to instruct unity to log its output to specific sub directories in the logs folder. This helper is intended for tools such as Jenkins which doesn't allow to override the file path when archiving artifacts.

        The value can be set in multiple ways (gradle properties, environment variable, parameter in code) The precedence order is:

        • direct parameter in code
        • gradle properties
        • environment variables
        • hardcoded value
        Returns:
        a category value
        Default:
        ""
        See Also:
        UnityPluginConsts.UNITY_LOG_CATEGORY_OPTION
        UnityPluginConsts.UNITY_LOG_CATEGORY_ENV_VAR

      • java.lang.Boolean getRedirectStdOut()

        Returns if the external unity command should redirect stdOut.

        By default Unity logs all output into a logfile. This property determines if the Unity log output will be redirected to stdout.

        If a custom logFile is provided, the output will be logged to the stdout and the logfile. This property works only on macOS.

        Returns:
        true if log should be redirected to stdout
        Default:
        false

      • java.io.File getUnityPath()

        Returns a File path to a Unity installation.

        The value can be set in multiple ways (gradle properties, environment variable, parameter in code) The precedence order is:

        • direct parameter in code
        • gradle properties
        • environment variables
        • hardcoded value
        The hardcoded default value is system dependent.
        See Also:

        UnityPluginConsts.UNITY_PATH_ENV_VAR
        UnityPluginConsts.UNITY_PATH_MAC_OS
        UnityPluginConsts.UNITY_PATH_MAC_OS
        UnityPluginConsts.UNITY_PATH_WIN
        UnityPluginConsts.UNITY_PATH_WIN_32
        UnityPluginConsts.UNITY_PATH_LINUX
        Returns:
        the path to unity executable
