            • java.util.Set<BuildTarget> getTestBuildTargets()

              Returns a java.util.Set of BuildTarget objects to construct unity editmode/playmode tasks.

              The plugin constructs a series of test tasks based on the returned java.util.Set.

              Example Test Task structure with two test build targets: 

               :check
 |--- :test
 +--- :testEditMode
 |    +--- :testEditModeAndroid
 |    |--- :testEditModeIos
 |--- :testPlayMode
      +--- :testPlayModeAndroid
      |--- :testPlayModeIos
 
     
              Returns:
              the buildtargets to generate test tasks for
              Default:
              the a set with the defaultBuildTarget
              See Also:
              UnityPluginConvention.defaultBuildTarget
